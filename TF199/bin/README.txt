README – Executable Information

Please note:

The project is configured to be run through the
Visual Studio solution file (TF199.slnx),
as required by the assignment instructions.

When the project is run using Visual Studio,
all assets such as sounds and music are loaded
correctly using relative file paths.

If TF199.exe is run directly by double-clicking,
some assets may not load due to differences in
the working directory used by the operating system.
This behaviour is expected and does not affect running
the project through Visual Studio.

To run the project:

1. Open TF199.slnx in Visual Studio
2. Build and run the project (Ctrl + F5 or F5).
